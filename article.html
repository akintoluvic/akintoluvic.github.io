{{// Outline}} // Implementing a Deep of an Object // Deep copying is when all the values

Given an object1 below, we will create object2 by assigning it the value of object1

        var object1 = {
            name: "Ale Ole",
            age: 12,
            single: true
        }

        var object2 = object1

Next, we will log object2 in console to show that it is an actual copy of object1 

        console.log(object2);  
        // console logs { name: "Ale Ole", age: 12, single: true }
